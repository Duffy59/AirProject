# Air project
## Informations
Technical test for leroy merlin

## How to install it
To install and run the application you have to :

- Install a jdk 11
- Install maven

 - run : 
```bash
maven clean package
```



## How to run it
To run the application you have to :

- Install docker
 
 - run : 
```bash
docker-compose build
```

 - and run : 
```bash
docker-compose up
```
